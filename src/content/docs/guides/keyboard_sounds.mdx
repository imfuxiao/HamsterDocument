iOS 系统中，音量分为两个部分：「铃声和提示音」和「媒体音量」，两者的音量是彼此独立的。

* 「媒体音量」是指播放音频或视频时的音量；

* 「铃声和提示音」是指系统提示音（如来电铃声）、通知音、按键音等。

因「按键音」的音量由「铃声和提示音」控制，当「铃声和提示音」的音量改变时，按键音的音量也会随之改变。

而「按键音」与「媒体音量」无关，所以当你播放音频或视频时，调整「媒体音量」时，按键音是不变的。

「用按钮调整」在「关闭」状态下:

* 「铃声和提示音」的音量只能通过截图中的划动条调整；
* 「媒体音量」的音量:

  1. 可以通过「物理音量键」调整；
  2. 「控制中心」中的「音量条」调整。

「用按钮调整」在「开启」状态下:

* 「铃声和提示音」可通过:

  1. 截图中的划动条调整；
  2. 当系统无音频或视频播放时，可通过「物理音量键」调整。

* 「媒体音量」的音量:

  1. 在有音频或视频播放时，可通过「物理音量键」调整；
  2. 「控制中心」中的「音量条」调整。

## 「仓」的按键音

仓的按键音也受「铃声和提示音」的音量控制，但有以下几种情况：

- 情况1：当音量在 `0 < 音量 <= 1` 的区间内时，表示按键音量为「铃声和提示音」的音量的百分比，如设置音量为0.2，表示为当前「铃声和提示音」音量的20%。
  此时，按键音受「铃声和提示音」的音量控制，当「铃声和提示音」的音量改变时，按键音的音量也会随之改变。

- 情况2(特殊情况)：当音量为0时，表示使用系统按键音，与系统键盘按键音逻辑一致，即按键音量不再受「铃声和提示音」的音量控制，可以始终保持在一个稳定的音量。

- 情况3：当「键盘皮肤」功能中设置了自定义按键音时，按键音的音量受「铃声和提示音」的音量控制，音量大小在 `0 <= 音量 <= 1` 之间。注意：此时按键音量设置为0，也不会转为情况2。

## 按键音相关问题

1. 已经开启按键音开关，但是没有按键音。

请检查当前系统是否为静音模式，静音模式下是没有按键音的。

2. 播放音频或者视频时，听不到按键音。

* 检查系统「铃声和提示音」的音量是否太低了，测试时可以调到最大；
* 检查「仓」中的「按键音」设置是否太低了，测试时可以调到最大；
* 也可以将「仓」中的「按键音」设置为0，然后在测试；

3. 按键音为什么要跟随「铃声和提示音」的音量变化？

因系统 API 限制，第三方按键音无法独立控制音量，只能跟随系统音量变化。

仓的按键音在 `0 < 音量 <= 1` 的区间内时，使用的是第三方按键音，所以按键音会跟随「铃声和提示音」的音量变化。

如果不能接受这个设定，可以将仓的键音设置为0，此时仓使用的是系统按键音，系统按键音不受「铃声和提示音」的音量控制，可以始终保持在一个稳定的音量。

4. 导入皮肤后，音量变了，如何处理？

导入皮肤后，请重新选择一下皮肤。
